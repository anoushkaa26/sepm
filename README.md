# 📚 StudySphere – Your AI-Powered Study Companion

StudySphere is a powerful, AI-integrated learning productivity app built with Flutter. Whether you're studying solo or collaborating with peers, this app helps boost your focus, manage time, and elevate your academic journey. From smart timers to personalized study assistants – it’s your all-in-one toolkit for smarter learning.

---

- Secure login with **Firebase Authentication** (Email & Password).
- Manage user profiles with customizable name, email, and avatar.
- Seamless onboarding with persistent login state.

## 🔧 Tech Stack
- **Frontend:** Flutter (v3.7.2)
- **Backend:** Firebase (Firestore, Auth)
- **Local Storage:** SQLite, Shared Preferences
- **AI & Tools:** Gemini AI (via API), YouTube Integration

## 📦 Key Packages
- `firebase_core`, `firebase_auth`, `cloud_firestore`
- `flutter_chat_ui`, `file_picker`, `sqflite`, `provider`

## 🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/jiya19g/sepm.git

# Install dependencies
flutter pub get

# Run the app
flutter run
```

**Note:** Ensure Firebase setup and platform-specific configuration (Android/iOS) are completed.
